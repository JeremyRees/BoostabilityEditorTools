Review code customization

When you activate the extension—either by clicking the icon in the top-right corner of the browser or via the keyboard shortcut (Ctrl+Shift+A by default)—you are prompted for a review code.

The first character of this code indicates whether to rate the article three, four, or five stars. For a three or four star rating, the remaining characters identify which criteria to suggest for the writer to improve on.

For example, using the default options, the code "djl" would produce the following review: "Great work. Your article has been accepted at 4 stars. In the future, please focus on taking a creative approach and strengthening your authority and expertise."

You can use the table below to customize the keys used to construct the review for an article.

Three Stars Four stars Five stars
Improve on: Examples Creative Approach -
Improve on: Flow Grammar -
Improve on: - Authority/Expertise -

Automation

In the future, the extension will be able to automatically complete the task after entering a review comment. It will also be able to check for the target keyword and insert a link if it's found as soon as the page loads.

These features are coming soon. Once they are implemented, you will be able to enable or disable them here.

Keyboard shortcuts

Many of the extension's features are accessed via keyboard shortcuts; the defaults are listed below. You can customize these shortcuts at "chrome://extensions/shortcuts"*. Just be careful that your customizations don't conflict with any existing Chrome shortcuts.

*Chrome doesn't allow linking directly to that page, but you can copy the address into a new tab to access it.

Function Shortcut
Open the review code prompt Ctrl + Shift + A
Pause or resume the task timer Ctrl + Shift + S
Insert the client link on the selected text Ctrl + Shift + Z
